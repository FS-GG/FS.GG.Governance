- [x] File and directory are typed distinctly.
- [x] Provenance, licence, method, destination, and digest are mandatory.
- [x] Tree identity is cross-platform, ordinal, raw-byte, and injective.
- [x] Missing/unreadable/symlink/path-escape/mutation cases fail closed.
- [x] Exemption follows verification.
- [x] Text-attribute policy is enforced.
- [x] Real filesystem tests cover the failure legs.
- [x] Consumer documentation and packaged source are present.
